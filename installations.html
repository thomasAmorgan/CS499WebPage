Installation

In order to work with the game in the current state all you need is a web browser (preferably Google Chrome) and an IDE (Atom, Netbeans, etc.). Simply navigate to the file in browser url field, for example file:///users/thomas/Documents/index.html. Since we do not have a server running our program you have to manual open the field in the browser in order to view and interact with it. This requires you to save any edits done in the IDE and then reload the page in order for your changes to take effect. Ideally it would be a good idea to implement a server/client system in order to better host the game.
