Future Enhancements/Maintaince

  • Future Enhancements for the Project

    One of the more vital future enhancements for future groups to accomplish is to add in the game rules implementation. The logic that Grayson and Tyler created, along with any pseudocode that was created for this section will be provided to the customer for future teams to use. The main focus of this enhancement will be adding in user interface functionality to the preexisting logic, because our team did not quite make it that far into development. Another possible future enhancement could be for an improved user interface. While our team did quite a good job with creating a serviceable user interface, improvements are also possible. A lot of the hex functionality is already implemented, but we haven’t reached the point where bot placement is implemented, so that could be a good starting point.

  • Issues Raised by Users During Product Testing That Required Fixes

    Unfortunately, our game was not far along enough in development to be distributed to users for testing. We only had the hex grid working, which does have interactivity but does not warrant user testing. This is something that future groups may want to look into when they get closer to a fully fleshed out game.

  • Project Features/Documentation Added to Aid Maintaince

    One of the key requests of our customer was to have fully documented code. To meet this request, we documented what our code did, why it was there, etc. Every bit of code that we wrote was fully documented as to make the code easily accessible for future development teams to build on and understand. Our code doesn’t really need maintenance necessarily, not this early in the development cycle, anyways, but we did have to make it easily understandable for future teams to be able to pick up and run with. This will circumvent a common issue with teams that get code that has already been started, which is that they don’t understand what was wrote, therefore they cannot build on it and decide to start over instead of moving forward with development.
