Requirements

General Gameplay

TENTATIVE

In its first state, this game will be a "hot seat" game. (A hot seat game is a game played by multiple people using one computer.) The minimum viable product for this game in priority order are as follows:

A Serviceable UI
  • A two-dimensional hexagonal grid play space
  • A representation for pieces on the board
  • A player turn order indicator
  • An action point indicator
  • A representation of cards
  • Correct user interaction with all elements
Rule Algorithms
  • Implementation of rules for combat and movement
  • Implementation of rules for game startup
  • Implementation of rules for board manipulation
  • Implementation of rules for player actions
  • Implementation of rules for card actions
