Test Plan

Scope

For each test, test in:
  • Chrome
  • Firefox
  • Internet Explorer
  • Microsoft Edge
For each test, test with grid size:
  • 20
  • 30
  • 60

Section 1: Individual functionality testing

Place Grid
  • Test: Place a grid on a valid hex.
  • Test: Attempt to place a grid on an invalid and unfilled hex (edge of screen)
  • Test: Place a grid on a valid hex. Attempt to place a grid on top of the first grid.
  • Test: Place grids all alongside the edges of the screen.
  • Test: Fill up the screen with grids.
Swap Grid
  • Test: Attempt to select an invalid (untiled) hex.
  • Test: Attempt to swap two validly placed grids.
  • Test: Select a grid, then deselect the grid.
  • Test: After selecting and deselecting a grid, reselect the same grid and attempt to swap with another grid.
  • Test: After selecting and deselecting a grid, select a different grid and attempt to swap with the originally deselected grid.
  • Test: After selecting and deselecting a grid, select a different grid and attempt to swap with a third, yet-untouched grid.
  • Test: Place two grids. Attempt to switch them, then switch them back to their starting positions.
  • Test: Place three grids. Attempt to swap grid 1 with grid 2, grid 2 with grid 3, and grid 3 with grid 1.
  • Test: Place four grids, one along each border outside of the screen. Attempt to switch them in any order.
  • Test: Place four grids, one in each corner of the screen. Attempt to switch them in any order.
Move Grid
  • Test: Attempt to select an invalid (untiled) hex.
  • Test: Place two grids, and attempt to move grid 1 to a valid hex outside of grid 2.
  • Test: Place two grids, and attempt to move grid 1 to an invalid hex.
  • Test: Select a grid, then deselect the grid.
  • Test: After selecting and deselecting a grid, reselect the same grid and attempt to move it to a valid location.
  • Test: After selecting and deselecting a grid, select a different grid and attempt to move it to a valid location.
  • Test: Move a grid to a valid location, then attempt to move the same grid to another valid location.
  • Test: Move a grid to a valid location, then attempt to move a different grid to a valid location.
  • Test: Place a grid where a moved grid would be allowed to be placed in a corner. Attempt to move a grid into the corner. Do this for each corner.
Rotate Grid
  • Test: Attempt to rotate an invalid (untiled) hex.
  • Test: Attempt to rotate a grid.
  • Test: Attempt to rotate a grid on each edge of the screen.
  • Test: Attempt to rotate a grid in each corner of the screen.

Section 2: Integrated functionality testing

S = Swap Grid
M = Move Grid
R = Rotate Grid
Test several action sequences as specified, using a mixture of boundary and middle cases:
  • S-M-R
  • S-R-M
  • M-S-R
  • M-R-S
  • R-S-M
  • R-M-S
  • S-M-R-S-M-R-S-M-R
  • S-S-S-M-M-M-R-R-R
  • Sx30
  • Mx30
  • Rx30
  • Random sequence with at least 30 steps
